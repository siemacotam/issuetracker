## General info

This is the rectuitment task - simple issue tracker.
User can :

- Add new issues (which app save in local storage)
- Sort issues by status
- Switch issue status
- Hide / show issue description

### Technologies

Project is created with :

- React and Typescript
- React Context
- Formik
- Styled components
